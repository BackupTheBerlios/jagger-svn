Jagger Homepage

Overview

Jagger is a Java application monitoring tool using JMX technology to aggregate, archive and visualize monitoring data for larger computer clusters, giving developers and administrators both a succinct and comprehensive view into their systems, which normal JMX consoles cannot do due to information overflow.

Major features

  • implemented in Groovy — flexible, yet fully integrated into J2EE environents.
  • console mode with tailored information panels about the state of your JVM, container and applications.
  • description of monitoring data model (PLANNED)
  • aggregation of distributed mbean data in clustered host setups, exporting the result as mbeans again (PLANNED)
  • archival of monitoring data (PLANNED)
  • SLA configuration, monitoring and enforcement (PLANNED)
